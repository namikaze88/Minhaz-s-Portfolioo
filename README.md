## 🚀 **Deployment**

### GitHub Pages (Recommended)
1. **Fork or clone** this repository
2. **Enable GitHub Pages** in repository settings
3. **Select source** as main branch
4. **Access** your portfolio at `https://yourusername.github.io/repository-name/`

### Local Development
```bash
# Clone the repository
git clone https://github.com/namikaze88/Minhaz-s-Portfolioo.git

# Navigate to project directory
cd Minhaz-s-Portfolioo

# Open in browser
open index.html

# Or use a local server
python -m http.server 8000
```

### Custom Domain
1. **Add CNAME file** to repository root
2. **Configure DNS** settings with your domain provider
3. **Enable custom domain** in GitHub Pages settings

## 🔧 **Customization**

### Personal Information
Update the content in `index.html`:
- **Hero section**: Name, title, company
- **Experience**: Work history and responsibilities
- **Skills**: Technical expertise and tools
- **Projects**: Personal projects and achievements
- **Contact**: Social media and contact information

### Styling
Modify `css/style.css`:
- **Colors**: Update CSS custom properties
- **Fonts**: Change font families in imports
- **Animations**: Adjust keyframes and transitions
- **Layout**: Modify grid and flexbox properties

### Content
Replace placeholder content:
- **Images**: Add your profile photos in `images/`
- **Project Images**: Update project screenshots
- **Links**: Update social media and project URLs
